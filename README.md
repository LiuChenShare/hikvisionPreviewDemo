# hikvisionPreviewDemo
海康威视视频实时预览Demo

# 相关技术及环境
VS2019 + Winform + .NET Framework 4

# 使用说明（v1.0.0.0）
调用海康威视固定ip端口的摄像头，实现相应功能  
目前已实现：
```
1、实时预览
2、录制视频
```
